## üìä Scoring System Explained

### How Components Are Scored

1. **Dimensional Measurement**: Each component measured on 4-5 dimensions
2. **Normalization**: Raw metrics converted to 0-100 scale
3. **Weighted Aggregation**: Dimensions weighted by importance
4. **Ensemble Fusion**: Components scored using category-specific weights
5. **Percentile Ranking**: Components ranked in percentiles
6. **Grade Assignment**: A+ to F grades assigned based on score
