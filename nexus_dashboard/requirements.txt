# NEXUS Dashboard - Requirements & Setup

## Python Version
Requires: Python 3.7+

## System Requirements
- Operating System: Linux, macOS, Windows (with WSL2)
- Terminal: Support for 256-color ANSI sequences
- Screen Resolution: Minimum 80x24, Recommended 120x40+

## Python Dependencies
ZERO external dependencies - all modules from Python standard library

### Standard Library Modules (all included in Python 3.7+)
- os - File system operations
- sys - System interface
- json - JSON serialization/deserialization
- time - Time functions
- random - Random number generation
- threading - Thread-based parallelism
- asyncio - Asynchronous I/O
- curses - Terminal control (TUI)
- dataclasses - Class decorator for data storage
- typing - Type hints and annotations
- enum - Enumeration support
- datetime - Date and time objects
- collections - Specialized container datatypes
- math - Mathematical functions

## Installation

### 1. Navigate to Dashboard Directory
```bash
cd /workspaces/.ZSHRC/nexus_dashboard
```

### 2. Verify Python Version
```bash
python3 --version  # Should be 3.7 or higher
```

### 3. Run Dashboard
```bash
python3 dashboard.py
```

### 4. Terminal Configuration (Optional)

For best visual effects, set terminal environment:

```bash
# Linux/macOS
export TERM=xterm-256color
export COLORTERM=truecolor

# Windows WSL2
set TERM=xterm-256color
```

## Configuration Files

### Registry Location
- **Default**: `~/.nexus/registry.json`
- **Created**: Automatically on first run
- **Format**: JSON with component metadata

### Example Registry Path
```bash
~/.nexus/registry.json
```

## Troubleshooting Setup

### Python Not Found
```bash
# Install Python 3.7+
sudo apt-get install python3  # Ubuntu/Debian
brew install python@3.11      # macOS
```

### Permissions Error
```bash
# Make script executable
chmod +x dashboard.py

# Or run with explicit interpreter
python3 dashboard.py
```

### Import Error for 'curses'
```bash
# curses is included in Python on Linux/macOS
# On Windows, use WSL2 or install via:
pip install windows-curses  # (Only if needed)
```

## Known Limitations

1. **Terminal Color Support**: Requires 256-color ANSI support
2. **Screen Size**: Minimum 80x24 recommended
3. **Unicode Support**: Full Unicode rendering required
4. **Refresh Rate**: Limited by terminal I/O speed (~100ms minimum)
