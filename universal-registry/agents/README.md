# Agents Sub-Registry (Universal Hyper Registry)

This directory defines an Agents Sub-Registry for advanced tools and agents:

- Vector stores (Weaviate/Redis)
- Web search
- Neo4j LLM Knowledge Graph Builder
- Git/Repo tools (GitPython)
- Advanced Crawling/Parsing/Scraping
- DAG/RAG++/GraphRAG with Ensemble Fusion
- Hyper Modality Multimodal
- Screen Capture (mss/node-mss)
- VLA Planner + Playwright Executor
- XAI (Explainability)
- Observability (Grafana/Prometheus bridges)

See `registry.py` for default registrations. Heavy dependencies are optional; uncomment in `requirements.txt` as needed.
