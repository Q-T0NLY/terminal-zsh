## ğŸ†˜ TROUBLESHOOTING

### Issue: Components not loading
**Solution:** Run `python3 nexus_ultra_installer.py` and select "1. View Main Dashboard"

### Issue: Dashboard won't start
**Solution:** Ensure Flask is installed: `pip3 install flask flask-cors psutil`

### Issue: Setup fails with permissions
**Solution:** Run with sudo or add user to docker group: `sudo usermod -aG docker $USER`

### Issue: Database connection fails
**Solution:** Run `python3 backend_config.py` and configure database properly

### Issue: Python packages not installing
**Solution:** Create virtual environment: `python3 -m venv .venv && source .venv/bin/activate`
